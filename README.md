# 🧠 Smart IT Helpdesk Assistant

An AI-powered IT troubleshooting assistant that helps users diagnose and resolve common technical issues step-by-step.

---

## 🚀 Overview

Smart IT Helpdesk Assistant is a full-stack application designed to simulate a real-world IT support experience.

Instead of giving generic answers, the assistant:
- Identifies the most likely cause of an issue
- Asks targeted questions
- Guides the user through troubleshooting
- Narrows down to a final diagnosis

---

## ✨ Features

- 💬 Chat-based troubleshooting interface
- 🧠 AI-powered diagnosis (OpenAI API)
- 🔍 Step-by-step issue isolation
- 🌙 Light / Dark mode
- ⚡ Real-time responses
- 🎯 Focus on practical, real-world IT problems

---

## 🛠️ Tech Stack

**Frontend**
- React (Vite)
- CSS (custom styling)

**Backend**
- Node.js
- Express

**AI**
- OpenAI API

---

## 🧩 How It Works

1. User describes a technical issue (e.g. "Wi-Fi not working")
2. Assistant identifies the most likely cause
3. Assistant asks clarifying questions
4. Troubleshooting steps are suggested only when needed
5. The issue is narrowed down to a root cause

---

## 📸 Demo

> Example use case:
- User reports Wi-Fi issue
